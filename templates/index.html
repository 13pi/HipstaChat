Open Source

Для работы чата используются только технологии с открытым исходным кодом.

Вы можете использовать в коммерческих и некоммерческих целях.

Для работы чата необходим Python c Django и база данных: PostgreSQL, либо SQLite.




Клиент-серверная архитектура. API

Чат состоит из двух частей: Frontend (AngularJS) и Backend (Python).

Для коммуникации чатсей этой системы используется REST.

Данные между клиентом и сервером передаются в формате JSON.
